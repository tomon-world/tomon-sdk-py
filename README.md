# Tomon-sdk
python version of Tomon sdk 

## Key Features:
* official Tomon API
* support *async* and *await*

## Install:
Python 3+ is required.

```
# Linux/macOS
python 3 -m pip install -U  tomon-sdk

# Windows
py -3 -m pip install -U tomon-sdk.
```
